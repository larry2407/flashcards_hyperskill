Implementing the EntrySet method Open on Hyperskill

The Java standard Hashtable class has a method called entrySet that allows users to iterate over the entries of a table. Your task here is to implement a simplified version of the method for the described hash table.

Input: the first line contains an integer $n$ -- the number of queries. Each of the next $n$ lines contains two records separates by space: the first represents an integer key, the other corresponds to a string value.

Output: for each unique id, print all entries associated with the id. All ids can be printed in arbitrary order, entries should be printed in the same order as they were inserted into a table.


Sample Input:
5
513 Alice
401 Bob
513 Mike
401 Kate
513 John
Sample Output:
513: Alice Mike John
401: Bob Kate


Memory limit: 256 Mb
Time limit: 5s
