Numbers filter Open on Hyperskill

Write a class with the name NumbersFilter that extends the Thread class and overrides method run . It should read integer numbers from the standard input (line by line). If a number is 0, the worker must stop; otherwise, if the number is even, the worker must print it to the standard output (each number on a new line).

Use the provided template for your class.

The testing system will start your class as a regular thread.

Sample Input:
1
2
3
4
0

Sample Output:
2
4


Memory limit: 256 Mb
Time limit: 5s
